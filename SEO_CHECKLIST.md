# SEO checklist

- Уникальные title и description для маршрутов
- Canonical URL
- Open Graph и Twitter Card
- `robots.txt`, `sitemap.xml`, manifest, favicon
- JSON-LD Organization, Product, ItemList
- Человекопонятные URL и хлебные крошки
- Один H1, последовательные H2-H3
- Alt-тексты у изображений
- Внутренняя перелинковка
- Уникальные описания категорий

Перед production необходимо установить реальный `NEXT_PUBLIC_SITE_URL` и
проверить домен в поисковых системах.
