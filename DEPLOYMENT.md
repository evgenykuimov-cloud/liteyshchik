# Развёртывание

## Vercel

1. Подключить GitHub-репозиторий.
2. Импортировать проект в Vercel.
3. Указать `NEXT_PUBLIC_SITE_URL` с production-доменом.
4. Настроить секреты почты, CRM и хранилища только после реализации адаптеров.
5. Выполнить production deployment.

## Проверки

```bash
npm run lint
npm run typecheck
npm run build
```

После развёртывания проверить форму, 404, sitemap, robots, мобильную навигацию,
адаптивность и корректность канонических адресов.
